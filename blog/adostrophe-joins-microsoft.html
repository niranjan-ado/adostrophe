Adostrophe Joins Microsoft for Startups Founders Hub

Azure App Service and CDN will enable us to create optimised, high-performance websites that drive results for our clients.

By Niranjan | Feb 26, 2025 | Microsoft

Adostrophe is working with the Microsoft for Startups Founders Hub, which provides resources to support our full-stack development. Access to Azure cloud services, Visual Studio Code, GitHub, and Azure DevOps helps us build scalable, secure, and efficient solutions tailored to client needs. We use Azure Active Directory and Key Vault to help protect user data and API keys, focusing on security from the start.

Web Development and SEO

We are migrating our website to a C# application hosted on Azure App Service, with content delivery optimized via Azure CDN. Following this migration, we plan to offer web development and SEO (Search Engine Optimization) capabilities, complementing our existing SEM services, including Google Ads and Google Ad Grants.

We are working to follow web.dev best practices, prioritizing performance, accessibility, and SEO. This includes semantic HTML, responsive designs, and advanced browser features. Performance is being optimized through code splitting, lazy loading with the Intersection Observer API, and image optimization using WebP and AVIF formats. WCAG compliance is being addressed through ARIA attributes and proper color contrast. Azure SQL Database and Azure Functions facilitate data management and serverless backend operations. Development and testing are being streamlined using Microsoft Copilot and Google Gemini. Websites are built with Blazor and .NET 8, hosted on the Azure platform. Input validation and SQL injection protection are implemented to help maintain data integrity.

Custom Matterport Virtual Tours

Building upon our existing Matterport virtual tour services, we are developing custom tours with improved accessibility for visually impaired users. Features include ARIA attributes, custom keyboard shortcuts, audio description scripts, and JavaScript for focus management. Integrating the Matterport API with Azure Cognitive Services, specifically Text-to-Speech, allows for the creation of virtual tours with audio guides. Our DPIIT recognition in AR/VR allows us to explore immersive technologies, including potential integration with Azure Spatial Anchors. Microsoft Copilot and Google Gemini are used for code optimization and debugging. We are exploring machine learning-driven security solutions for Matterport tours, utilizing Azure Security Center and Azure Sentinel.

A Mobile Game

We are developing a mobile game focused on improving financial literacy, designed for accessibility across diverse communities. The game tracks daily income and expenses, providing feedback through rewards and warnings. Machine learning generates financial reports presented in a user-friendly dashboard. Recognizing the prevalence of limited-capability smartphones and intermittent internet, we are prioritizing performance optimization and robust offline functionality. This ensures users can track their finances and access crucial information even with limited or no connectivity.

Technically, the game utilizes a modular design for scalability and maintainability. Financial literacy modules are implemented with lightweight, efficient data storage, enabling offline access and seamless synchronization upon reconnection. This is achieved through a local, encrypted database, ensuring data confidentiality. Synchronization with the backend is incremental, minimizing data transfer and bandwidth consumption.

The game covers financial concepts like passive income strategies, entrepreneurship, insurance, and cybersecurity, drawing information from government sources. To improve engagement, we are integrating ancestral information and location-specific scenarios using Google Street View and Matterport APIs. This allows users to interact with financial situations within familiar environments. Development is underway for iOS and Android, with plans for AR/VR integration, including Apple Vision Pro. We are using ARKit on iOS for environmental mapping, motion tracking, and virtual object placement. We are also exploring Azure Spatial Anchors to enable location-based AR experiences, allowing users to return to virtual financial scenarios in real-world locations. This integration aims to provide interactive learning experiences, making financial education more engaging.

The game aims to become a daily tool for users, supporting financial habits from an early age. We are interested in open-source development, with the codebase available on GitHub to encourage international collaboration and localization. This will enable developers to adapt the game to meet the needs of diverse communities. A DevOps pipeline powered by Azure Monitor and Azure DevOps supports deployment of updates. Azure services are used for user authentication, data management, and insights.

By using Azure solutions, we are working to strengthen our capabilities and deliver solutions to modern challenges. This collaboration helps us create value for our clients and the community.
